Installation

Manual installation

Reference the file in your project:

<script src="jquery.resizestart.js"></script>

Using Bower

bower install jquery.resizestart

Using NPM

npm install jquery.resizestart

Usage

Basic usage.

Direct method reference:

// Default options
$(selector).resizestart(function() {
	// execute callback...
});

// Custom options
$(selector).resizestart(1000, function() {
	// execute callback...
});

Event binding:

// Default options
$(selector).on('resizeend', function() {
	// execute callback...
});

// Custom options
$(selector).on('resizeend', 1000, function() {
	// execute callback...
});

Options

All plugin & event options.

Option Type Default Description
delay integer/
object		 250 The amount of time, in millisecond, to wait after the last resize event was fired before triggering the event.
